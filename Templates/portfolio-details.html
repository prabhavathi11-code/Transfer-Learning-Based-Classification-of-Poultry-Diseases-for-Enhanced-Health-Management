Poultry Disease Detection System

This project focuses on the early detection of poultry diseases using deep learning and image classification. A custom-trained CNN model identifies diseases such as Coccidiosis, Newcastle Disease, Salmonella, and differentiates them from healthy chickens.

The system includes a web interface built with Flask and Bootstrap. Users can upload images of chickens, and the model predicts the likely disease condition, helping farmers take early action and prevent outbreaks.

Tools Used: TensorFlow, Keras, Flask, Bootstrap, Roboflow (for dataset preparation).
